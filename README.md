# Manage API

Short description and motivation.

## 安装方法

1. 添加以下内容到Gemfile

```ruby
gem "manage-api"
```

2. 运行以下命令

```bash
$ bundle
$ rails manage_api:install
$ rails manage_api_engine:install:migrations
```
